安装完成后, 点开Crack压缩包, 将R2018b和\verb+license_standalone.lic+提取到
\verb+/Downloads/MATLAB+下:
\begin{figure}[H]
    \centering
    \includegraphics[scale=0.4]{./figures/matlabinstall/8.png}
\end{figure}

将\verb+license_standalone.lic+移动到安装目录下:

\verb+ sudo cp -f ~/Downloads/MATLAB/license_standalone.lic/usr/local/MATLAB/R2018b+

将解压的R2018b下的所有文件复制到安装目录下\verb+/usr/local/MATLAB/R2018b+

\verb+ sudo cp -rf ~/Downloads/R2018b/* /usr/local/MATLAB/R2018b/+

修改目录权限, 否则会出现许多warning:

\verb+ sudo chmod -R 777 /usr/local/MATLAB/+

运行matlab:

\verb+ sh /usr/local/MATLAB/R2018b/bin/matlab+

没有错误则成功.

接下来创建快捷方式文件:

\verb+sudo gedit /usr/share/applications/Matlab.desktop+

在弹出的文本框里写入以下内容:
\begin{verbatim}
[Desktop Entry]
Type=Application
Name=Matlab 2018b
GenericName=Matlab 2018b
Exec=sh /usr/local/MATLAB/R2018b/bin/matlab -desktop
Icon=/usr/local/MATLAB/R2018b/toolbox/shared/dastudio/resources/MatlabIcon.png
Terminal=false
Categories=Development;Matlab;
\end{verbatim}

\section{高分辨率屏幕调整}
\verb+ >> s = settings;s.matlab.desktop.DisplayScaleFactor

\verb+ >> s.matlab.desktop.DisplayScaleFactor.PersonalValue = 1.5
